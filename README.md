# Exercício – Concorrência em Django

Este exercício demonstra o uso de concorrência em Django
utilizando a biblioteca threading.

## Implementação
- Execução síncrona de tarefas simuladas com time.sleep
- Execução concorrente utilizando threads
- Comparação do tempo de execução

## Como testar
1. Execute o servidor:
   python manage.py runserver
2. Acesse:
   /sincrona/
   /concorrente/
3. Compare os tempos retornados.
